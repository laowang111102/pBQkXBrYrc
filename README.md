# 项目简介 python106

本项目是一个基于Python语言开发的水果销售系统，采用MySQL数据库进行数据存储，并以Django框架为基础进行搭建。

## 技术栈

- **后端语言**: Python
- **开发框架**: Django
- **数据库**: MySQL

## 功能模块

系统主要包括以下功能模块：

- 用户管理
- 商品管理
- 订单管理
- 报告统计

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.8+
- MySQL 5.7+
- 操作系统：支持Linux、Windows

## 部署步骤

1. 环境准备：安装Python环境、MySQL数据库
2. 克隆项目代码
3. 安装依赖：使用pip安装`requirements.txt`中列出的依赖包
4. 数据库配置：创建数据库，导入数据表结构
5. 运行项目：使用命令行启动Django应用

## 目录结构

```
project/
│
├── apps/                 # 应用目录
│   ├── user/             # 用户模块
│   ├── product/          # 商品模块
│   └── order/            # 订单模块
│
├── db/                   # 数据库配置
├── media/                # 媒体文件存储
├── static/               # 静态文件
├── templates/            # 模板文件
│
├── manage.py              # 管理脚本
├── requirements.txt       # 依赖列表
└── settings.py            # 配置文件
```

- **高效率开发**: 采用Django框架，提高开发效率，降低维护成本。
- **灵活扩展**: 系统模块化设计，便于扩展和二次开发。
